                              Mini skype

                    Alternative Skype client by Popov Evgeniy Alekseyevich

                             Version 0.2.6


System requirement

Microsoft Windows 10 (22H2) or higher
An internet connection

Description and usage

This program is an alternative client for web version of Skype.
